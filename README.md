

#INVOICE APP 

Invoice app built with laravel and Vue.js

## Clone App
```
gh repo clone mpete5993/Invoice-App
```

## Install npm dependancy
```
npm install
```

## Run database migration
```
php artisan migrate
```

## Run database seeder
```
php artisan db:seed
```
## Run laravel server
```
php artisan serve
```

## Start npm server
```
npm run dev
```
